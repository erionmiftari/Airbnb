# Airbnb App

Airbnb App është një aplikacion për menaxhimin dhe rezervimin e banesave.  
Përdoruesit mund të krijojnë llogari, të shtojnë banesat që i kanë për qira, dhe të rezervojnë banesa për 1 natë ose më shumë.

## Features

- Regjistrimi dhe login i përdoruesve (JWT authentication)
- Shto, modifiko dhe fshi listings (banesa)
- Shfaqja e listings me filtrime për lokacion, çmim dhe date
- Rezervimi i banesave me validim të disponueshmërisë
- User profile me historikun e rezervimeve dhe listings
- Responsive UI (desktop + mobile)

## Teknologjitë

- **Frontend:** React + TailwindCSS
- **Backend:** Node.js + Express
- **Database:** PostgreSQL
- **Authentication:** JWT
- **Version Control:** Git + GitHub
